# LINE Harness ビジネス概要書

## これは何？

LINE公式アカウントの **CRM・マーケティング自動化ツール** です。
L-step（月額2,980円〜）や Utage（月額9,700円〜）と同等の機能を、オープンソース（無料）で提供します。

**一言で言うと**: 「LINE公式アカウントを、プログラマブルに自動運用できる基盤」

## 主要機能一覧

### 1. 友だち管理

| 機能 | 説明 |
|------|------|
| 自動登録 | 友だち追加時にWebhookで自動登録（名前・アイコン取得） |
| タグ付け | 自由にタグを作成・付与してセグメント管理 |
| カスタムメタデータ | 任意のキー・値ペアでユーザー情報を拡張 |
| セグメント | タグ・スコア・日時の組み合わせで絞り込み |

### 2. メッセージ配信

| 機能 | 説明 |
|------|------|
| ステップ配信 | 友だち追加やタグ付与をトリガーに、時間差で自動配信 |
| 一斉配信 | 全員 or セグメント指定で一括送信。予約配信対応 |
| リマインダー配信 | 指定日からの逆算で「3日前」「当日」等に自動通知 |
| 自動応答 | キーワードマッチで即時返信（完全一致 / 部分一致） |
| オペレーターチャット | 人間が介入して個別対応できるチャット機能 |

**対応メッセージ形式**: テキスト / 画像 / Flex（カード型） / カルーセル / ボタン / クイックリプライ

### 3. 計測・分析

| 機能 | 説明 |
|------|------|
| CV（コンバージョン）計測 | 購入・申込み等のゴールを定義し、達成を記録 |
| URLクリック計測 | 短縮URLでクリック数・クリックユーザーを追跡 |
| 流入経路追跡 | `?ref=instagram` 等のパラメータでどこから来たか記録 |
| 広告連携 | Google Ads（gclid）、Facebook Ads（fbclid）、UTMパラメータ自動取得 |
| アフィリエイト | パートナー別のクリック・CV・売上を集計。コミッション計算 |
| リードスコアリング | ユーザー行動にポイント付与し、見込み度を数値化 |

### 4. 自動化（IF-THEN ルール）

イベント発生時に自動でアクションを実行：

| トリガー（IF） | アクション（THEN） |
|----------------|---------------------|
| 友だち追加 | タグ付与 / シナリオ登録 |
| メッセージ受信 | 自動返信 / Webhook通知 |
| タグ変更 | リッチメニュー切替 |
| スコア閾値到達 | メッセージ送信 |
| CV発火 | タグ付与 / シナリオ登録 |
| カレンダー予約 | リマインダー登録 |

### 5. データ収集

| 機能 | 説明 |
|------|------|
| LIFFフォーム | LINE内で動くWebフォーム（テキスト/メール/電話/日付/選択式等） |
| 回答時自動アクション | フォーム送信でタグ付け・シナリオ登録を自動実行 |
| リッチメニュー | 6ボタン / 3ボタンのビジュアルメニュー。ユーザー別に切替可 |

### 6. 外部連携

| 連携先 | 内容 |
|--------|------|
| Stripe | 決済完了 → 自動タグ付け・CV記録・スコア加算 |
| Googleカレンダー | 予約受付 → LINE通知・リマインダー自動設定 |
| Webhook（送信） | イベント発生時に任意のURLへPOST（Zapier/Make等と連携可） |
| Webhook（受信） | 外部からLINE Harnessの機能を呼び出し |

## アーキテクチャ概要（非エンジニア向け）

```
┌──────────────┐     ┌──────────────────┐     ┌──────────┐
│  LINEユーザー  │────▶│  Cloudflare      │────▶│ データベース│
│  （友だち）     │◀────│  Workers (API)   │◀────│ (D1/SQLite)│
└──────────────┘     └──────────────────┘     └──────────┘
                            │    ▲
                            │    │
                     ┌──────▼────┴──────┐
                     │   管理画面        │
                     │  (Next.js)       │
                     └──────────────────┘
```

**ポイント**:
- **サーバーレス**: サーバー管理不要。Cloudflareが全部やってくれる
- **顧客データはLINE社が管理**: DB上に保持するのは LINE UserID・表示名・アイコンURL 程度
- **認証基盤不要**: LIFF側はLINEログインで完結。管理画面はAPIキー認証のみ
- **画像ストレージなし**: 画像はLINEサーバーまたは外部URL参照（S3/R2等は別途検討）

## 独自の強み

### 1. AI-native設計
全機能がREST APIで操作可能。Claude等のAIエージェントが直接CRMを操作できる。
「人間が監視し、AIが運用する」が設計思想。

### 2. アカウントBAN対策（UUID自動引継ぎ）
LINEアカウントがBANされても、新アカウントに友だちデータ（タグ・シナリオ進捗・スコア）を自動引継ぎ。
業界初の機能。UUID（内部ID）でLINE UserIDに依存しない管理を実現。

### 3. ステルスモード
配信タイミングにランダムなゆらぎを加え、一斉配信を自然な送信パターンに偽装。
バッチ処理の間隔調整・レート制限（自主規制1,000通/分）でBAN検知リスクを低減。

## まとめ

LINE Harnessは **「L-step/Utageの機能を、ほぼ無料で、自社管理できる」** オープンソースツールです。

**向いているケース**:
- LINE公式アカウントの運用コストを下げたい
- APIで自由にカスタマイズ・外部連携したい
- AIエージェントに運用を任せたい
- アカウントBAN時のリスクヘッジが必要
- 複数アカウントを統合管理したい

**検討が必要なケース**:
- 画像の管理・配信を多用する（別途ストレージ導入が必要）
- 技術リソースがゼロ（初期セットアップにエンジニアが必要）
- 100万人超の大規模運用（DB移行の検討が必要）
